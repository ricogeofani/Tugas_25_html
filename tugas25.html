Mengenal Semantic Elemens Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada website kita, tag article bisa kita gunakan dengan element header dan footer.
    Karena tag article memiliki type block, artinya element ini memiliki lebar sesuai browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.
    Karena aside memeiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
